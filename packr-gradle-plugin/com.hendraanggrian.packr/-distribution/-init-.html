Distribution(platform: Platform, name: String, jdk: String? = System.getenv("JAVA_HOME") ?: System.getProperty("java.home"))

Represents a platform-specific configuration.

Parameters

platform - target platform of this distribution.

name - file name of this distribution that will be generated. Default is project's name.

jdk - directory, ZIP file, or URL to ZIP file of an OpenJDK or Oracle JDK build containing a JRE used to build this distribution. Default is Java Home environment variable, if any.
