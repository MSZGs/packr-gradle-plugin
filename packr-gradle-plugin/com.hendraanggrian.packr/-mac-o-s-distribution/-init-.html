MacOSDistribution(projectDir: File, name: String, jdk: String? = System.getenv("JAVA_HOME") ?: System.getProperty("java.home"), icon: File? = null, bundleId: String? = null)

Represents a macOS distribution configuration, providing extra properties only relevant in macOS.

Parameters

projectDir - working directory of org.gradle.api.Project needed to point relative paths.

name - file name of this distribution that will be generated. Default is project's name.

jdk - directory, ZIP file, or URL to ZIP file of an OpenJDK or Oracle JDK build containing a JRE used to build this distribution. Default is Java Home environment variable, if any.

icon - Location of an AppBundle icon resource (.icns file) relative to project directory. This is an optional property.

bundleId - The bundle identifier of your Java application, e.g. com.my.app. This is an optional property.
