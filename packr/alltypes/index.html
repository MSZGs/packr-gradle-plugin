All Types

com.hendraanggrian.packr.PackTask

Task that will generate native distribution on each platform.
com.hendraanggrian.packr.PackrConfiguration

Base configuration for packr task and extension.
com.hendraanggrian.packr.PackrExtension

Packr main extension, modifications of this POJO will be applied to all PackTask. It is not mandatory to modify this extension, configuring each PackTask will result in same behavior.
com.hendraanggrian.packr.PackrPlugin

When PackrPlugin is applied, several instances of this PackTask with different platform name will be created.
