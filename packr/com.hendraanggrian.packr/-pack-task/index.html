Location of an AppBundle icon resource (.icns file). This is an optional property.

mainClass

 var mainClass: String

The fully qualified name of the main class, using dots to delimit package names. Must be defined or will throw an exception.

minimizeJre

 var minimizeJre: String

Minimize the JRE by removing directories and files as specified by an additional config file. Comes with a few config files out of the box. Default is soft.

openOnDone

 var openOnDone: Boolean

Open outputDir upon packing completion. This is an optional property.

outputDir

 var outputDir: File?

The output directory. Default is release directory in project's build directory.

outputName

 var outputName: String?

The output name. Default is project's name.

resources

 var resources: MutableList<File>

List of files and directories to be packaged next to the native executable. Default is empty.

verbose

 var verbose: Boolean

Print extra messages about JRE minimization when set to true. This is an optional property.

vmArgs

 var vmArgs: MutableList<String>

List of arguments for the JVM, without leading dashes, e.g. Xmx1G. Default is empty.

wrapApp

 var wrapApp: Boolean

Append app file extension to mac distribution. Default is true.
